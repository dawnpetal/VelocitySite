Tutorial 03

Cloud Scripts

Browse ScriptBlox and rscripts.net without leaving Velocity. Search, read the source, then execute or save locally.

01

Open the Script Hub

Click the cloud icon in the left activity bar to open the Script Hub panel. Velocity indexes two sources: ScriptBlox and rscripts.net, and makes both searchable from a single interface without opening a browser.

02

Search and filter

Type a game name or keyword in the search bar at the top of the panel. Results update live. Use the source toggle to filter by ScriptBlox, rscripts.net, or show both at once. Results are sorted by most recent by default.

Search is debounced at 300ms. Type and wait a beat rather than hammering Enter.
03

Preview before executing

Click any result to expand it. The full source loads in a read-only Monaco view inside the panel, syntax highlighted and scrollable. Read it before you do anything else. Then hit Execute to run immediately, or Save to copy it into your workspace first.

Always read scripts before running them. Public hubs are unmoderated. If you don't understand what a script does, don't execute it.
04

Save, edit, and make it yours

Hit Save on any cloud script and it lands in your workspace with its original filename. From there it's a local file, open it in the editor, modify it, and execute your version with Cmd + Return.
