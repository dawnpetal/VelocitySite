Tutorial 01

Getting Started

Install VelocityUI, pick your executor, write and execute your first script. Zero to running in four steps.

01

Install VelocityUI

Run the install command from your terminal. The script pulls the latest release and drops it in /Applications.

Terminal zsh 
curl -fsSL https://raw.githubusercontent.com/dawnpetal/VelocityUI/main/install.sh | bash
Requires macOS 12+ and jq. All ship with modern macOS.
02

Launch and pick your executor

Open VelocityUI from /Applications. On first launch you'll land on the Settings tab. Under Executor, choose either Hydrogen or Opiumware, then hit Reload.

VelocityUI is a UI layer only. It does not include an executor, you need Hydrogen or Opiumware separately and running before you can execute scripts.
03

Write and execute your first script

Switch to the Editor tab. Type or paste a script into the Monaco editor, then press Cmd Return OR the Run Button to execute. The status bar at the bottom shows injection state in real time.

Editor lua 
print("Hello from VelocityUI")
local Players = game:GetService("Players")
local lp = Players.LocalPlayer
print(lp.Name)
04

Save to your workspace

Press Cmd + S to save. The file is written to your workspace and appears immediately in the sidebar file tree. Use the sidebar's new-file button or right-click context menu to create additional files and folders without leaving the app.
